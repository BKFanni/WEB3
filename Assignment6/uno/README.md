## Setup/Running
1. git clone https://github.com/BKFanni/WEB3.git
2. cd WEB3/Assignment6/uno
3. npm install
4. Set up environment variables:
    * JWT_SECRET="secret used for signing JWT sessions"
    * MONGO_URI="mongoDB database connection URI"
5. npm run build
6. npm start
